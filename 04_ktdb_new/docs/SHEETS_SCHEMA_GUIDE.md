# KTDB Google Sheets 스키마 가이드

> Streamlit 분석 앱(`streamlit_app.py`)과 연동되는 Google Sheets 데이터 구축 규격서

## 1. 전체 구조

앱은 4개의 Google Sheets 스프레드시트를 데이터 소스로 사용합니다.

| # | 데이터 | Secrets 키 | 용도 |
|---|--------|-----------|------|
| 1 | 사회경제지표 | `SHEET_URL_SOCIO` | 인구, 취업자, 종사자 등 지역별 통계 |
| 2 | 목적OD | `SHEET_URL_OBJ_OD` | 출근/등교/업무/귀가/기타 통행량 |
| 3 | 주수단OD | `SHEET_URL_MAIN_OD` | 승용차/버스/지하철/철도 수단별 통행량 |
| 4 | 접근수단OD | `SHEET_URL_ACC_OD` | 접근수단별 통행량 |

---

## 2. 시트별 스키마 명세

### 2-1. 사회경제지표 (`SHEET_URL_SOCIO`)

이 스프레드시트에는 **8개 탭**이 필요합니다.

#### 존체계 탭 (마스터)

| 탭 이름 | `존체계(행정구역)` |
|---------|-------------------|

| 컬럼 | 타입 | 설명 | 예시 |
|------|------|------|------|
| `SIDO` | 문자열 | 시도명 | 서울특별시 |
| `SIGU` | 문자열 | 시군구명 | 종로구 |
| `ZONE` | 정수 | 250존 번호 (1~250) | 1 |

- 행 수: **250행** (헤더 제외)
- 이 탭은 OD 데이터의 존번호→지역명 변환에 사용되므로 **반드시 존재해야 합니다**

| 탭 이름 (정확히 일치) | 데이터 내용 |
|-----|------|
| `총 인구수` | 총인구 |
| `5-24세 인구수` | 5~24세 인구 |
| `15세이상 인구수` | 15세 이상 인구 |
| `취업자수` | 취업자 |
| `수용학생수` | 학생 |
| `종사자수` | 종사자 |

| 컬럼 | 타입 | 설명 | 예시 |
|------|------|------|------|
| `SIDO` | 문자열 | 시도명 | 서울특별시 |
| `SIGU` | 문자열 | 시군구명 | 종로구 |
| `ZONE` | 정수 | 250존 번호 | 1 |
| `2023` | 숫자 | 2023년 값 | 141860 |
| `2025` | 숫자 | 2025년 값 | 135964 |

## 3. 필수 규칙

### 3-1. 컬럼 헤더

- **반드시 영문 코드 사용**: `SIDO`, `SIGU`, `ZONE`, `ORGN`, `DEST`, `2023` 등
- 한글 헤더 사용 금지 (앱 내부에서 영문→한글 변환 처리함)
- 연도 컬럼은 **숫자만**: `2023` (O) / `2023년` (X)

### 3-2. 데이터 값

- **천단위 콤마 금지**: `19800` (O) / `19,800` (X)
- Google Sheets 셀 서식을 "자동" 또는 "일반 텍스트"로 설정
- 소수점은 허용: `135963.8`

### 3-3. 탭 이름

- **정확히 일치해야 함** (대소문자, 공백, 괄호 포함)
- 예: `목적OD (2023년)` — 괄호 앞 공백 1개, "년" 포함

### 3-4. 빈 컬럼/행

- 데이터 영역 밖에 **빈 열이 있으면 삭제**
- 원본 XLSX 업로드 시 L~Q열 등 빈 열이 생길 수 있으므로 확인 필요

## 4. 원본 XLSX에서 업로드하는 방법

### 원본 파일 위치

```
04_ktdb_new/docs/
├── 전국_지역간_사회경제지표.xlsx    → SHEET_URL_SOCIO
├── 2024-OD-PSN-OBJ-00_전국권.xlsx  → SHEET_URL_OBJ_OD
├── 2024-OD-PSN-MOD-10_전국권.xlsx  → SHEET_URL_MAIN_OD
└── 2024-OD-PSN-MOD-20_전국권.xlsx  → SHEET_URL_ACC_OD
```

### XLSX ↔ Google Sheets 탭 매핑

| XLSX 시트명 | Google Sheets 탭명 |
|-------------|-------------------|
| **사회경제지표** | |
| ZONE | 존체계(행정구역) |
| POP_TOT | 총 인구수 |
| POP_YNG | 5-24세 인구수 |
| POP_15P | 15세이상 인구수 |
| EMP | 취업자수 |
| STU | 수용학생수 |
| WORK_TOT | 종사자수 |
| **목적OD** | |
| PUR_2023 | 목적OD (2023년) |
| PUR_2025 | 목적OD (2025년) |
| ... | ... |
| **주수단OD** | |
| MOD_2023 | 주수단OD (2023년) |
| ... | ... |
| **접근수단OD** | |
| ATTMOD_2023 | 접근수단OD (2023년) |

### 업로드 절차

1. Google Sheets에서 대상 스프레드시트 열기
2. 해당 탭 선택 (없으면 새로 만들기 — 탭 이름 정확히 입력)
3. A1 셀에서 시작하여 XLSX의 해당 시트 데이터를 붙여넣기
4. **확인사항:**
   - 첫 행이 영문 헤더인지 확인
   - 숫자에 천단위 콤마가 안 붙었는지 확인 (서식 > 숫자 > "일반 텍스트")
   - 데이터 뒤 빈 열이 없는지 확인

## 5. 검증 체크리스트

- [ ] `존체계(행정구역)` 탭: 3컬럼(SIDO, SIGU, ZONE), 250행
- [ ] `총 인구수` 탭: 10컬럼(SIDO~2050), 250행, 숫자에 콤마 없음

- [ ] `15세이상 인구수` 탭: 동일 구조

- [ ] `수용학생수` 탭: 동일 구조
