## [1페이지] 개발 가능성 평가

### 1. 요청 사항 정리

도로부에서 제안한 **4대 핵심 모듈**은 다음과 같습니다.

| 순번 | 모듈명 | 기능 | 우선순위 |
|---|---|---|---|
| 1 | **주상도 해석** (Borehole Log Parser) | PDF 주상도에서 지층명·심도·N치·지하수위·RQD/TCR 추출 | **우선 개발** |
| 2 | **데이터 통합** (Data Matcher) | 물리시험·역학시험 결과를 시추공번+심도 기준으로 매칭·통합 | **우선 개발** |
| 3 | 지반정수 산정 및 그래프 | 설계 분야별 지반정수 산정, N치 분포·e-logP 등 그래프 자동 생성 | 후순위 (1,2 완료 후) |
| 4 | 이상치 탐지 | 공학적 범위를 벗어난 시험값 자동 필터링 및 경고 | 후순위 (1,2 완료 후) |

**처리 흐름**: 지반조사 보고서(GIR) PDF 입력 → 4대 모듈 처리 → 정형화된 엑셀, 물성치 산정 결과, 분석 그래프 출력

---

### 2. 핵심 과제 — 왜 어려운가

설명자료에서 제시된 핵심 과제는 다음과 같습니다.

- 서로 다른 포맷(이미지, 텍스트, 표)의 데이터가 **하나의 PDF에 혼재**
- **심도(Depth)를 Key**로 여러 시험 결과를 매핑해야 함
- **업체별로 주상도 양식이 다르고**, 수치 입력 양식도 상이
- 표 형태가 복잡하고, 깊이(Y축)에 따라 데이터가 나열되는 특수한 구조

기존 문자 인식 소프트웨어는 텍스트만 읽을 수 있어, 주상도의 지층 기둥 그래픽·색상 마킹·복합 표 구조를 처리할 수 없었습니다. 2024~2025년부터 상용화된 **이미지를 이해하는 AI 기술**이 이 한계를 처음으로 돌파하였습니다.

### 3. 기술 검증 결과 — NTB-24 시추주상도 실증 테스트

수도권광역급행철도 B노선 제4공구 시추주상도 NTB-24 (4페이지, 70m)를 Google Gemini AI로 실제 분석하였습니다.

**Module 1 (주상도 해석) 검증 결과:**

| 추출 항목 | 설명자료 Data Schema | 추출 결과 | 정확도 |
|---|---|---|---|
| 지층명 | 매립토, 퇴적토, 풍화토, 풍화암, 연암 등 | ✔ 성공 | 높음 |
| 심도 (Depth) | 지층이 변하는 깊이 (m) | ✔ 성공 | 높음 |
| N치 (N-Value) | 30cm 관입 타격 횟수, 정수형 | ✔ 성공 | 높음 |
| 지하수위 | 지하수 발견 깊이 (m) | ✔ 성공 | 높음 |
| RQD / TCR | 암반 품질 지표 (%, 풍화암 이하) | △ 부분 성공 | 중간 |
| 파쇄대 위치 | 취약구간 심도 | △ 부분 성공 | 중간 |
| 터널 구간 마킹 (빨간색) | 설계 구간 표시 | ✔ 성공 | 높음 |

**Module 2 (데이터 통합) 검증 전망:**

| 대상 데이터 | 설명자료 Data Schema | AI 인식 가능성 |
|---|---|---|
| 물성시험 총괄표 (Summary Table) | Wn, Gs, LL/PI, 입도분포 | 높음 — 정형 표 구조 |
| SOIL TEST DATA | 물리시험 결과 | 높음 — 표 인식 용이 |
| ROCK TEST DATA | 역학시험 결과 | 높음 — 표 인식 용이 |
| UCS Laboratory Results | 일축압축강도 (qu) | 높음 — 정형화된 양식 |
| PLT Laboratory Results | 점하중시험 결과 | 중간 — 양식 다양 |
| BH Soil Basic Properties | 종합 물성표 | 높음 — 요약표 형태 |

> **종합 판단**: Module 1·2 모두 **개발 가능** (실증 완료 / 전망 양호)
> - 핵심 데이터 추출 정확도: **90% 이상** 추정
> - 보조 데이터: **70~85%** — 엔지니어 검수로 보완 가능

### 4. 최종 활용 — 6대 설계 분야 연계

개발 완료 시 아래 설계 검토에 필요한 입력 데이터를 자동 정리할 수 있습니다.

| 설계 분야 | 핵심 입력 변수 | 데이터 소스 |
|---|---|---|
| **비탈면 (토사)** | 단위중량(γ), 점착력(c), 지하수위 | 물리시험, 삼축/직접전단시험 |
| **비탈면 (암반)** | 불연속면 방향, 암석강도(qu), RQD, RMR/SMR | 현장 페이스매핑, 일축압축시험 |
| **기초 (말뚝/얕은기초)** | N치, 점착력(c), 내부마찰각(φ), 변형계수(E) | 표준관입시험, 공진재하시험 |
| **흙막이 (가시설)** | 단위중량, 점착력, 변형계수(Es, kh) | 물리시험, 현장시험 |
| **터널** | 탄성계수(E), 포아송비(v), σc, RQD | 암석역학시험, 시추코어분석 |
| **연약지반** | 압축지수(Cc), 선행압밀하중(Pc), 압밀계수(Cv) | 압밀시험, 물리시험 |

### 5. 비용 및 한계

**AI 서비스 이용료**: 시추 1공(4페이지) 분석에 약 **1~5원**

| 규모 | 월 처리량 | 이용료 (월) |
|---|---|---|
| 소규모 | 100공 (400페이지) | 약 1,000 ~ 3,500원 |
| 중규모 | 500공 (2,000페이지) | 약 5,500 ~ 18,000원 |
| 대규모 | 2,000공 (8,000페이지) | 약 22,000 ~ 72,000원 |

**한계 및 유의사항:**

- 현재 기술로 완전 무인 자동화는 어려우나, **작업 시간을 크게 단축**하는 보조 도구로 즉시 활용 가능

### 방법 1 (추천): Google Gemini — 무료, 가장 간단

**준비물:** 구글 계정

```
1단계: gemini.google.com 접속 → 구글 계정으로 로그인
2단계: 화면 하단 클립 아이콘 → 시추주상도 PDF 또는 시험성적서 PDF 첨부
3단계: 아래 프롬프트를 복사하여 붙여넣기 후 전송
4단계: AI 응답 결과 확인 → 표를 복사하여 엑셀에 붙여넣기
5단계: 원본과 대조하여 검수
```

**복사용 프롬프트 — 주상도 해석용 (Module 1):**

```
첨부한 시추주상도(Drill Log) PDF에서 다음 데이터를 추출하여
엑셀에 바로 붙여넣을 수 있는 표 형식으로 정리해주세요.

1. 기본정보: 시추공번, 위치좌표, 시추표고, 시추심도, 지하수위
2. 지층구성: 지층명, 심도(상단~하단), 두께, 토질분류(USCS)
3. SPT N치: 측정심도, N값, 관입불능(반발) 여부
4. 암반구간: TCR(%), RQD(%), 파쇄대 위치 및 특성
```

**복사용 프롬프트 — 시험 데이터 통합용 (Module 2):**

```
첨부한 지반조사 시험성적서에서 다음 데이터를 추출하여
시추공번 + 채취심도(Depth) 기준으로 정리된 표를 만들어주세요.

1. 물리시험: 함수비(Wn), 비중(Gs), 액성한계(LL), 소성지수(PI), 입도분포
2. 역학시험: 일축압축강도(qu), 점착력(c), 내부마찰각(φ)
3. 각 항목의 지층별 평균, 표준편차를 함께 표기해주세요.
```

### 방법 2: Google AI Studio — 무료, 반복 작업에 강력

**준비물:** 구글 계정 (Gemini와 동일)

AI Studio(aistudio.google.com)는 Gemini와 같은 AI이지만, **프롬프트를 저장해두고 반복 사용**할 수 있어 과업 내 여러 시추공을 처리할 때 훨씬 효율적입니다.

```
1단계: aistudio.google.com 접속 → 구글 계정으로 로그인
2단계: 좌측 "Create Prompt" → "Freeform" 선택
3단계: 상단 "System Instructions"에 입력:
       ─────────────────────────────
       당신은 지반조사 전문가입니다.
       입력된 시추주상도 또는 시험성적서에서
       데이터를 추출하여 정해진 표 형식으로 정리합니다.
       심도(Depth)를 기준으로 데이터를 매핑하세요.
       항상 한국어로 응답하세요.
       ─────────────────────────────
4단계: 본문에 PDF/이미지 첨부 + 복사용 프롬프트 붙여넣기
5단계: "Run" 클릭 → 결과를 엑셀에 복사
6단계: 다음 시추공 → 파일만 교체하여 "Run" 다시 클릭
```

**Gemini 대비 장점:**
- 프롬프트 저장/재사용 → 같은 과업 내 수십 공을 효율적으로 반복 처리
- 파일 크기 한도 훨씬 넓음 (2GB vs Gemini 100MB)
- AI 모델 직접 선택 가능 (Gemini 2.5 Flash / Pro)
- "Get Code" 기능으로 자동화 개발 연동 가능

### [참고] 자동화 시스템 개발 시 추가 가능한 기능

Module 1·2 개발 완료 후, 연계하여 아래 기능을 추가 구현할 수 있습니다.

- 수십~수백 공 **일괄 처리** (파일 업로드 → 자동 분석)
- 서식 적용된 **엑셀 자동 생성** (업체별 양식 대응)
- **지층별 N치 분포 그래프** 및 평균 N치 자동 출력
- 시험결과 **지층별 통계** (평균, 표준편차, 분산) 자동 산출
- 지반정수 자동 산정 및 **이상치 자동 경고** (Module 3·4)

AI Studio의 "Get Code" 기능으로 자동화 기초 코드를 생성할 수 있으며, 이를 기반으로 개발팀이 본격적인 시스템을 구축할 예정입니다.
