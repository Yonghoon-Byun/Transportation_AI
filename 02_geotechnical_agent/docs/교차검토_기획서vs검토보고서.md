| 번호 | 항목 | 기획서 원안 | 기술 검토 결과 | 영향도 | 조치 필요 여부 |

## 4. 기획서에서 누락된 항목 (검토 과정에서 추가 발견된 사항)

기획서가 다루지 않았으나, 기술 검토 및 실무 적용 과정에서 반드시 결정해야 할 사항들이다.

| 누락 항목 | 내용 | 중요도 |
|-----------|------|--------|
| 운영 비용 | 외부 AI 서비스 이용에 따른 월정 사용료가 발생한다 (과업 건수에 따라 월 10만~150만원 범위). 기획서에 비용 항목이 전혀 없음 | 높음 |
| 개발 기간 현실화 | 기획서 원안 약 2.5개월 → 검토 결과 약 4개월 필요. 1.5개월 추가 소요 | 높음 |
| 검증용 데이터 사전 확보 | 실제 지반조사 PDF 10건 이상(3개 이상 업체 양식 포함)과 수작업 정답 데이터가 개발 착수 전에 필요. 기획서에 준비 항목 없음 | 높음 |
| 지반공학 전문가 참여 계획 | 알고리즘 검증을 위해 실무 경력 5년 이상 전문가 1명이 개발 초기부터 참여해야 함. 기획서에 인력 계획 없음 | 중간 |
| 그래픽형 주상도 처리 한계 | 표 형태의 PDF는 잘 처리되지만, 그래픽 레이아웃의 주상도나 오래된 스캔 문서는 오류가 잦음. 수동 입력 경로 별도 마련 필요 | 중간 |

## 5. 조치 권고 요약

착수 전 반드시 완료해야 할 항목을 중요도 순으로 정리한다.

**1순위 — 데이터 보안 정책 확인 (착수 불가 블로커)**
외부 AI 서비스로 데이터를 전송하는 구조가 발주처 보안 정책과 충돌하는지 과업별로 사전 확인한다. 특히 관공서 발주 과업에서는 적용 가능 여부를 착수 전에 명확히 해야 한다. 기획서에 명시된 "사내 독립 서버 완전 차단" 방침은 현재 구현 방식으로는 충족되지 않음을 공식 인지해야 한다.

**2순위 — 목표 수치 단일화 및 단계별 목표 재설정**
기획서 내 80% vs 70% 불일치를 해소하고, 1차 완성 목표(50~60% 단축)와 6개월 이후 목표(70% 단축)를 분리하여 단계별 KPI로 재정립한다.

**3순위 — 개발 범위 경계 명확화**
HWP 자동 생성, 종단주상도 드림 연계, 고도화 기능(미조사 구간 예측 등)을 1차 범위에서 공식 제외하고, 향후 단계 과제로 별도 관리한다.

**4순위 — 착수 전 데이터·인력 확보**
실제 지반조사 PDF 10건 이상 확보, 수작업 정답 데이터 작성(약 2주), 지반공학 전문가 참여 확보를 개발 일정에 선행 조건으로 명시한다.

**5순위 — 운영 비용 예산 반영**
외부 AI 서비스 사용료(개발 기간 4개월 포함, 월 수십만원 수준)를 과업 예산에 반영한다. 과업 규모에 따라 변동이 크므로 범위(월 10만~150만원)를 기준으로 보수적으로 계획한다.
