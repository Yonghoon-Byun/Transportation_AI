## 검토 요약

기획서에서 제시한 핵심 기능(요약표·총괄표 자동 추출, 지반정수 산정, 그래프 출력)은 기술적으로 구현 가능하며 개발 착수에 무리가 없다. 다만 스캔 이미지 기반 PDF의 인식 정확도, 보안 방침과 외부 AI 서비스 사용의 양립 불가, 고도화 항목의 현실성 문제는 착수 전에 반드시 협의·조정이 필요하다.

---

## 1. 기술적으로 가능한 항목

| 항목 | 판정 근거 | 비고 |
|------|-----------|------|
| 디지털 PDF에서 표 형태 데이터 자동 추출 | 요약표·총괄표 구조가 일정한 경우 높은 정확도 달성 가능 | 스캔본 제외 |
| 시추공번호 + 심도 기준 데이터 매칭 | 키 필드(Hole No., Depth)가 명확하면 자동 매칭 처리 가능 | |
| USCS / AASHTO 자동 분류 | 입력값(소성지수, 입도 등)이 정형화되면 규칙 기반 분류 적용 가능 | |
| 경험식 기반 지반정수 자동 산정 | 6개 설계분야 입력변수 목록 기 정리 → 수식 구현 가능 | |
| N치 분포 등 표준 그래프 6종 자동 생성 | 정형 데이터가 확보되면 자동 작도 가능 | |
| 엑셀 요약표 자동 출력 | 가장 구현 용이한 성과품 형식 | 1차 산출물로 적합 |
| 공학적 범위 기반 이상치 탐지 및 경고 | 분야별 허용 범위 기준값 정의 후 자동 플래그 가능 | |
| 전문가 검증 루프 (수정·보완 기능) | 추출 결과 화면 확인 및 수동 수정 경로 설계 가능 | 필수 기능으로 권장 |

## 2. 기술적으로 불가능하거나 현 시점에서 어려운 항목

| 항목 | 사유 | 대안 |
|------|------|------|
| 한글(HWP) 파일 자동 생성 | HWP 포맷은 외부 도구 지원이 매우 제한적 | Word 형식으로 출력 후 수동 변환 |
| 스캔본 PDF에서 95% 추출 정확도 달성 | 이미지 기반 문서는 자동 문자 인식 오류가 빈번, 실측 60~70% 수준 | 스캔본은 수동 보완 경로 병행 필수 |
| 미조사 구간 지층·물성 예측 (고도화 항목) | 현재 AI 기술로 지반공학적 신뢰성 확보 어려움 | 1차 개발 범위에서 제외 후 별도 R&D 과제로 분리 |
| 최적 기초 형식 및 개량공법 자동 제안 (고도화 항목) | 설계 판단 영역으로 AI 단독 결정은 책임 소재 문제 발생 | AI 보조(후보 제시) 수준으로 기능 한정 |

## 3. 기획자와 협의가 필요한 항목

| 항목 | 기획서 내용 | 협의 필요 사유 | 선택지 |
|------|-------------|----------------|--------|
| 데이터 보안 방침 | "사내 독립 서버, 외부 유출 원천 차단" | 외부 AI 서비스를 사용하면 데이터가 외부 서버로 전송됨 → 두 방침은 양립 불가 | (A) 외부 AI 서비스 활용 + 내부 보안 기준 별도 수립 / (B) 외부 서비스 미사용 + 개발 범위 축소 |
| KPI 수치 불일치 | 섹션1 "80% 절감", 섹션3 "70% 단축" | 목표치 기준이 상이하면 완료 기준 합의 불가 | 단일 목표치로 통일 필요 |
| AI 학습 방식 표현 | "라벨링 데이터셋 구축" | 자체 AI 모델 학습은 수년 단위 투자가 필요한 작업임 → 기획 의도가 외부 AI 활용인지 자체 학습인지 불명확 | 표현을 "외부 AI 서비스 활용 고도화"로 정정 권장 |
| 종단주상도 드림 연계 | 설명자료에서 별도 언급 | 기획서 개발 범위에 미기재 → 포함 여부 미결정 | 1차 범위 포함 여부 명시 필요 |
| 1차 개발 대상 설계분야 | 6개 분야 전체 | 전체 동시 구현 시 개발 기간 장기화 | 우선순위 2~3개 분야 선택 후 순차 확장 권장 |

## 4. 기술적 한계점 (사전 인지 필요)

1. **양식 다양성**: 발주처·업체별 주상도 양식이 50종 이상으로 추정되며, 신규 양식은 출시 후 지속 추가 작업이 필요하다.
2. **파싱 정확도의 문서 품질 의존성**: 인쇄 상태가 불량하거나 수기 기입이 혼재된 PDF는 자동 추출 실패율이 높으며, 수동 입력 경로가 반드시 확보되어야 한다.
3. **초기 정확도**: 디지털 PDF 기준 1차 완성 시 약 85% 정확도 예상. 95% 목표는 실 데이터 검증 및 보완을 거쳐 6개월 이후 단계적 달성이 현실적이다.
4. **개발 기간**: 기획서에 미기재되어 있으나 6개 분야 전체 범위 기준 약 4~5개월 소요 예상.
5. **외부 AI 서비스 사용 시 운영비**: 보안 방침 협의 결과에 따라 월 수십만 원 수준의 이용료가 지속 발생할 수 있다.
6. **착수 전제 조건**: 실 업무에서 사용된 지반조사 PDF 10건 이상과 지반공학 전문가 1명의 상시 참여가 착수 전 확보되어야 한다. 해당 조건 미충족 시 개발 일정에 영향이 발생한다.
