## 1. 분석 개요

### 1.1 분석 목적

KTDB(국가교통 데이터베이스)에서 배포하는 OD(Origin-Destination) 원본 데이터의 파일 구조, 인코딩, 컬럼 구성, 샘플 데이터를 세밀하게 분석하여 PostgreSQL 적재 파이프라인 설계 및 ETL 로직 구현의 근거 자료를 마련한다.

### 1.2 분석 범위

| 구분 | 내용 |
|------|------|
| 분석 대상 | 2024년 기준 OD 데이터 ZIP 22건 + 과거년도(2023년 기준) 3건 = 총 25건 |
| 데이터 유형 | 목적별 OD (PSN-OBJ), 수단별 OD (PSN-MOD-1x), 목적별 주수단 OD / 접근수단 OD (PSN-MOD-2x), 화물 OD (FRE-CAR) |
| 권역 구분 | 전국 250존, 수도권, 부산울산권, 대구광역권, 대전세종충청권, 광주광역권, 제주권 |
| 파일 포맷 | .OUT, .TXT (CP949, 공백 구분자), .xlsx (Excel), .hwp/.hwpx (설명자료) |

### 1.3 핵심 발견 사항 요약

- **파일 포맷이 권역마다 다름**: 수도권은 텍스트 파일(.OUT, .TXT), 비수도권은 Excel(.xlsx), 전국 250존은 Excel(.xlsx)
- **수단 분류 체계가 3가지로 상이**: 수도권(10종 무헤더), 비수도권(9종 + 파생 2종), 전국 250존(7종)
- **존코드 체계가 3가지**: 수도권 7자리 문자열, 비수도권 정수형 1~N, 전국 250존 1~250
- **컬럼명 불일치**: 같은 수단도 `지하철` vs `도시철도` vs 위치 기반 미명명 등 명칭이 권역별로 다름
- **수도권 수단 TXT는 헤더 없음**: col5~col14의 10개 수단 컬럼이 무엇인지 설명자료 확인 전까지 불명확

### 2.1 분류 코드 체계

```
{연도}-OD-{대분류}-{소분류}-{권역코드}
  ↑       ↑          ↑          ↑
2024   PSN(여객)  OBJ(목적)  00(전국)
       FRE(화물)  MOD(수단)  01(수도권)
                            02(부산울산권)
                            03(대구광역권)
                            04(대전세종충청권)
                            05(광주광역권)
```

- **PSN-OBJ**: 목적별 여객 OD (출근/등교/업무/귀가/기타)

## 3. ZIP 내부 폴더 구조

ZIP 내부 폴더 구조는 권역에 따라 3가지 패턴으로 나뉜다.

### 3.1 수도권 패턴 (OBJ-01, MOD-11, MOD-21)

```
{ZIP명}/
├── 1. 존체계/
│   └── 존체계.xlsx
├── 2. 설명자료/
│   └── ★2024년_수도권OD및네트워크설명자료_수정_251020_최종.hwp
├── 3. 사회경제지표/
│   ├── 인구수/          SUB_POP{YY}.TXT         (7개: 23, 25, 30, 35, 40, 45, 50)
│   ├── 종사자수/        WORK_POP{YY}.TXT        (7개)
│   ├── 취업자수/        EMP_POP_{YY}.TXT        (7개, 언더스코어 주의)
│   └── 학생수/          STU_POP{YY}.TXT         (7개)
└── 4. OD/
    └── (데이터 파일들 — 유형별로 다름, 4절 참조)
```

**사회경제지표 파일명 규칙**:
- `SUB_POP{YY}.TXT` — 인구수 (예: SUB_POP23.TXT)
- `WORK_POP{YY}.TXT` — 종사자수
- `EMP_POP_{YY}.TXT` — 취업자수 (파일명에 언더스코어 하나 더 있음, 주의)
- `STU_POP{YY}.TXT` — 학생수

### 3.2 비수도권 패턴 (OBJ-02~06, MOD-12~16)

```
{ZIP명}/
├── 1. 존체계/
│   └── 존체계.xlsx
├── 2. 설명자료/
│   └── 2024년_전국 교통수요 분석 기초자료 설명자료.hwpx
├── 3. 사회경제지표/
│   └── {권역명}_사회경제지표.xlsx       ← 수도권과 달리 단일 xlsx 파일
├── 4. OD/
│   └── {권역명}_{목적통행|주수단통행}.xlsx
└── 5. 재차인원/
    └── 승용차, 버스 재차인원.xlsx
```

**수도권과의 주요 차이**:
- 사회경제지표가 TXT 개별 파일이 아닌 단일 xlsx 파일로 제공됨
- OD 폴더 번호가 4번 (수도권과 동일)
- 5번 폴더에 재차인원 데이터 추가

### 3.3 전국 250존 패턴 (OBJ-00, MOD-10)

```
{ZIP명}/
├── 1. 존체계/
│   └── 250 존체계.xlsx               ← 파일명이 "250 존체계"로 다름
├── 2. 설명자료/
│   └── 2024년_전국 교통수요 분석 기초자료 설명자료.hwpx
├── 3. 사회경제지표/
│   └── 전국지역간_사회경제지표.xlsx
├── 4. 재차인원/                       ← 4번이 재차인원 (OD가 아님, 주의)
│   └── 승용차, 버스 재차인원.xlsx
├── 5. OD/                            ← 5번이 OD (비수도권과 번호 다름)
│   └── 전국_2023년~2050년 {목적별|주수단별}OD(250).xlsx
└── 6. 연평균 일통행량(AADT) 전환계수/
    └── 2022년 수단별 주중 및 연평균 통계자료_OD배포 부가자료.xlsx
```

**비수도권과의 주요 차이**:
- 존체계 파일명이 `250 존체계.xlsx` (앞에 "250 " 접두어)
- 폴더 번호가 다름: 4번=재차인원, 5번=OD (비수도권은 4번=OD, 5번=재차인원)
- 6번 폴더에 AADT 전환계수 추가

### 3.4 화물 OD 패턴 (FRE-CAR-00)

```
{ZIP명}/
├── 배포용 (기준년도 2023년) 화물자동차OD_2025.11.28.xlsx
├── 배포용 장래년도 화물자동차OD_2025.11.28.xlsx
└── 화물OD 설명자료.hwp
```

**특이사항**: 다른 ZIP과 달리 하위 폴더 구조 없이 파일만 3개 있음.

## 4. 데이터 유형별 상세 분석

### 4.1 목적별 OD — 수도권 (.OUT 파일)

#### 파일 정보

| 항목 | 내용 |
|------|------|
| 위치 | `4. OD/ODTRIP{YY}_F.OUT` |
| 파일 수 | 7개 (연도: 23, 25, 30, 35, 40, 45, 50) |
| 인코딩 | CP949 |
| 구분자 | 공백 (가변 폭, space-delimited) |
| 헤더 | 없음 (1행부터 데이터) |
| 확장자 | .OUT |

#### 컬럼 구조 (9개)

| 위치 | 컬럼명 (추정) | 의미 | 타입 | 샘플값 |
|------|-------------|------|------|--------|
| col1 | O_seq | 출발존 순번 | int | 1 |
| col2 | O_zone | 출발존 코드 (7자리) | str | 1101072 |
| col3 | D_seq | 도착존 순번 | int | 1 |
| col4 | D_zone | 도착존 코드 (7자리) | str | 1101072 |
| col5 | 출근 | 출근 통행량 (통행/일) | float | 1293.031 |
| col6 | 등교 | 등교 통행량 (통행/일) | float | 205.414 |
| col7 | 업무 | 업무 통행량 (통행/일) | float | 464.115 |
| col8 | 귀가 | 귀가 통행량 (통행/일) | float | 1112.201 |
| col9 | 합계 | 합계 (통행/일) | float | 3949.478 |

#### 샘플 데이터 (ODTRIP23_F.OUT 첫 3줄)

```
1 1101072 1 1101072 1293.031 205.414 464.115 1112.201 3949.478
1 1101072 2 1101053 773.66 646.652 68.285 754.723 2629.308
1 1101072 3 1101054 866.019 253.771 128.15 749.279 2626.035
```

#### 특이사항

- `기타` 목적 컬럼이 없음 (비수도권 및 전국 250존에는 `기타` 있음)
- 합계 = 출근 + 등교 + 업무 + 귀가 (기타 미포함이므로 합계가 4개 목적의 합)
- 존코드가 7자리 문자열 (예: `1101072` = 서울시 종로구 청운효자동)
- 순번(seq)과 존코드(zone)가 함께 제공됨 — 순번은 행렬 인덱스, 존코드는 실제 행정코드

### 7.2 주요 차이점 요약

1. **도보/자전거, 택시 누락 (전국 250존)**: 전국 250존에는 도보/자전거와 택시가 없고, 대신 항공/해운이 별도 분리되어 있다.
